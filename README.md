A minimal, beginner-friendly website demonstrating the fundamentals of HTML, CSS, and JavaScript working together.

## Overview

This project is a simple single-page website designed as a learning resource for web development beginners. It showcases how HTML provides structure, CSS handles styling, and JavaScript adds interactivity. The page displays a random image, styled text, and a button that dynamically changes the paragraph text colour using JavaScript DOM manipulation.

## Features

- Clean, responsive HTML5 page structure
- Custom CSS styling with smooth transitions and modern button design
- JavaScript-powered interactive text colour toggle
- Random image fetched from [Picsum Photos](https://picsum.photos)
- Separated concerns with dedicated `scripts/` and `styles/` directories

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A text editor or IDE for making changes (optional)

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/danielcregg/simple-website-html-css-js.git
   ```
2. Navigate into the project directory:
   ```bash
   cd simple-website-html-css-js
   ```

### Usage

Open `index.html` directly in your browser, or use a live server extension in your editor.

The page will display a heading, a random image, a styled paragraph, and a button. Click the button to toggle the paragraph text colour between red and blue.

**Live Demo:** [https://danielcregg.github.io/simple-website-html-css-js/](https://danielcregg.github.io/simple-website-html-css-js/)
